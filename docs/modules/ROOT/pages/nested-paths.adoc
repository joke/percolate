== Source paths descend through accessors

A source path like `user.company.address.city` resolves one segment at a time, starting from the method
parameter (`user`). Each segment is matched to an accessor — a JavaBean getter (`getCity()`), a record-style
or fluent accessor (`city()`), or a public field — and the hops are chained: `user.getCompany().getAddress().getCity()`.

== Target paths nest construction

A target path like `location.city` tells percolate the value belongs to a nested object. It groups targets
by level and builds the intermediate objects, here constructing `new Location(...)` to hold the `city`
before passing it to `new Profile(...)`. Several directives that share a prefix (`location.city`,
`location.country`, …) assemble the same intermediate once.
