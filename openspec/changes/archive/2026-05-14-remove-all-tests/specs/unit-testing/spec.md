## REMOVED Requirements

### Requirement: Every processor class SHALL have a unit test
**Reason**: All existing test files are being removed as part of this change.
**Migration**: New unit test requirements will be defined in a follow-up change.

### Requirement: Unit tests SHALL use Spock mocking
**Reason**: All existing test files are being removed. The new testing framework is not yet defined.
**Migration**: New testing framework choice will be defined in a follow-up change.

### Requirement: New processor classes SHALL have unit specs
**Reason**: All existing test files are being removed.
**Migration**: New unit test requirements will be defined in a follow-up change.

### Requirement: Graph-stage classes SHALL have unit specs
**Reason**: All existing test files are being removed.
**Migration**: New unit test requirements will be defined in a follow-up change.

### Requirement: Pipeline unit spec SHALL cover the new stages
**Reason**: All existing test files are being removed.
**Migration**: New unit test requirements will be defined in a follow-up change.

### Requirement: Single-aspect graph-shape specs SHALL exist
**Reason**: All existing test files are being removed.
**Migration**: New integration test requirements will be defined in a follow-up change.

### Requirement: Golden DOT specs SHALL guard the rendering pipeline
**Reason**: All existing test files are being removed, including golden `.dot` files.
**Migration**: New golden test approach (if any) will be defined in a follow-up change.

### Requirement: Spock Groovy extension module SHALL exist for graph assertions
**Reason**: All existing test files are being removed, including the extension module.
**Migration**: New assertion helpers will be defined in a follow-up change.
